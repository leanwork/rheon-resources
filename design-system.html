Design System

Referência de identidade visual da Rheon. Paleta preto e branco, tipografia Inter (substituta de Rumiko para uso web), interface densa e tokens semânticos em OKLCH. Construído sobre a mesma arquitetura white-label do projeto base.

White-label: cada tenant pode sobrescrever cores, raios e tokens via API de layouts. Este documento exibe o tema base Rheon definido em src/styles.css.

Cores

Tokens semânticos em OKLCH. A Rheon opera sobre paleta preto e branco — preto para força e impacto, branco para equilíbrio e respiro visual. Use sempre os nomes semânticos em vez de valores fixos.
