# RHEON — Recursos para Agências

Repositório de materiais de referência para agências parceiras utilizarem como
base de implementação da plataforma de ecommerce da RHEON.

## Conteúdo

- **Design System** — referência visual com tokens semânticos, cores, tipografia,
  botões, badges e componentes.

## Desenvolvimento local

Abra `index.html` no navegador ou sirva com qualquer servidor estático:

```bash
npx serve .
```
